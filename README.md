# 🔐 Password Checker - Secure Password Generator & Security Checker
## 🌐 Live Demo - [👉 Click Here to Use Password Forge](https://password-generator-eosin-sigma.vercel.app/)

A powerful, client side password generation and security auditing tool that helps you create strong, uncrackable passwords and analyze existing ones for vulnerabilities.

---

## ✨ Features

### 🔑 Password Generator
- **Customizable length**: 4-128 characters (slider + manual input)
- **Character type selection**: 
  - Lowercase letters (always included)
  - Uppercase letters
  - Numbers (0-9)
  - Special symbols (!@#$%^&*...)
- **Real-time strength meter** with visual feedback
- **One-click copy** to clipboard
- **Secure randomization** using cryptographic methods
- **Instant regeneration** with animation feedback

### 🛡️ Security Checker
- **Password strength analysis** (Weak → Strong)
- **Entropy estimation** (bits calculation)
- **Pattern detection** (repeated chars, sequences, common words)
- **Character type breakdown** (Lower/Upper/Digit/Symbol)
- **Security recommendations** and warnings
- **Real-time feedback** with detailed analysis

### 🎨 User Experience
- **Modern glass-morphism design**
- **Fully responsive layout** (works on mobile, tablet & desktop)
- **Smooth animations** and visual feedback
- **Tab-based interface** (Generator / Security Checker)
- **No external dependencies** - 100% vanilla JavaScript
- **Works offline** after first load
